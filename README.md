# Javascript-Level-3
## 🚀 Tic Tac Toe Game (HTML, CSS & JavaScript)
<img src="https://github.com/user-attachments/assets/2c32d011-98ae-483f-b611-7be1bb005074" width="300" />


**🎮 Two Player Mode.**

Game supports two players (X and O) who take turns playing.

**💻 Built With Core Web Technologies.**

This project is built using only:
HTML
CSS
JavaScript
(No external libraries used)

**🖱️ Interactive UI.**

Click-based gameplay.
Simple and smooth user experience.

**🧠 Game Logic Features.**

Turn switching between players.
Winner detection system.
Draw (tie) detection.

**🏆 Winner Display**

When a player wins, the result is shown clearly on the screen.

**🔄 Restart Feature.**

A restart button appears after the game ends so players can play again easily.

**🎨 Modern UI Design.**

Clean interface.
Hover effects.
Simple and user-friendly design.

**🧼 Clean Code Structure.**

Code is organized and easy to understand.
